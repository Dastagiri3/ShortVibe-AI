# 🚀 ShortVibe-AI
---

🎬 AI-powered short video generation tool that transforms ideas, scripts, or long content into engaging short-form videos (Reels / Shorts / TikTok).

---

## 🌐 Live Demo

👉 https://shortvibe-ai.vercel.app

---

## 📌 Overview

ShortVibe-AI is an intelligent content automation tool designed to help creators, developers, and businesses generate viral short videos using AI.

It automates the entire pipeline:
👉 Script → Voice → Visuals → Captions → Final Video

---

## ✨ Features

- 🧠 AI Script Generation
- 🎙️ Text-to-Speech Voiceovers
- 🎥 Automated Video Creation
- 🖼️ Image / Video Asset Integration
- 📱 Vertical (9:16) Video Formatting
- 🔤 Auto Captions / Subtitles
- ⚡ Fast Processing Pipeline
- 🌍 Multi-language Support

---

## 🏗️ Architecture

User Input (Topic / Script)
↓
AI Processing Layer
↓
Script Generator (LLM)
↓
Voice Generator (TTS)
↓
Media Fetching (Images/Videos)
↓
Video Composer (FFmpeg / MoviePy)
↓
Final Short Video Output 🎬

## ⚙️ Installation

### 1. Clone the repository

git clone https://github.com/Dastagiri3/ShortVibe-AI.git

cd ShortVibe-AI


---

### 2. Install dependencies

For Python:


pip install -r requirements.txt


For Node.js:


npm install


---

### 3. Setup Environment Variables

Create a `.env` file:


OPENAI_API_KEY=your_api_key
GEMINI_API_KEY=your_api_key
TTS_API_KEY=your_api_key


---

### 4. Run the project

For Python:


python app.py


For Node.js:


npm run dev
